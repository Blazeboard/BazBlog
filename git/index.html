docker

查找镜像：docker search hello-world

获取镜像：docker pull

运行镜像：docker run

查看运行的容器：docker ps docker ps -a

操作容器：docker stop/start/rm

删除镜像：docker rmi

删除所有容器：docker rm $(docker ps -a -q)

删除所有镜像：docker rmi $(docker images -q)
